A customizable widget for list tweets from a user. Widget output messages are in portuguese (BR).

== Description ==

Now you can put in your template a list of tweets from a user easily. To change widget visual just go in /css in plugin folder and make changes that you want.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1.	Unzip and Upload br-twitter-widget folder to your /wp-content/plugins/ directory.
2. 	Activate the plugin through the 'Plugins' menu in WordPress.
3. 	Add the widget to your sidebar from Appearance -> Widgets and configure the widget options.
